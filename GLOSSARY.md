# Glossary

## Audio Processing
- **Sample Rate (SR)**: The number of audio measurements taken per second (e.g., 44,100Hz).
- **Bit Depth**: The precision of each audio measurement (e.g., 16-bit).
- **Mono/Stereo**: Mono is 1 audio channel; Stereo is 2 channels. This project uses Mono for simplicity.
- **Spectral Gating**: The technique of "muting" specific frequencies where noise is detected.

## Library Terms
- **Stationary Noise**: Noise that stays constant (like a fan or hum).
- **Non-Stationary Noise**: Noise that changes (like birds or traffic).
- **Time Masking**: Smoothing the transitions in time to prevent "chirping" sounds.
